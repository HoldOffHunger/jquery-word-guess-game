100% JQuery Word-Guessing Game

(Hangman, Except no References to the Death Penalty)

Demo :

http://www.earthfluent.com/word-guess-game-demo.html

Set it up using a minimal amount of syntax :

	$(document).ready(function(event) {
		var word = 'helloworld';
		WordGuessGame(word);
	});
